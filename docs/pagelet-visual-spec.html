② 美术规范 token

项规范说明
笔触粗细 stroke-width: 1.6px（主轮廓）/ 1.4px（表情线条） 足够 retina 清晰，又不喧宾夺主
线端 / 拐角 stroke-linecap: round · stroke-linejoin: round 圆头 = 手绘质感，避免机械感
手绘抖动 顶点坐标做 ±0.1~0.3 的微偏移（如 10.2 8.3 / 36.1 14.2 / 38.1） 不规则但克制，区别于"完美直线"
填充 fill: none（默认） 极简线稿（D004），保持"轻"
动效曲线 cubic-bezier ease-in-out · 2.4s 浮动 · 1.4s 脉冲 · 4s 眨眼 不规则节奏 = "呼吸感"
reduce-motion 关闭所有 animation；颜色仍随状态变化 D007 a11y 兼容
SVG 视窗 viewBox="0 0 44 44"，渲染尺寸 16-84px 自适应 同源缩放无失真

④ 实现要点

关注点实现对应章节

aria-live Suggestion 容器 aria-live="polite" SDD §9.2
