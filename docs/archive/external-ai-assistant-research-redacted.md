# External AI Assistant Research Redacted

This archive entry replaces a historical public comparison note that named a
specific external commercial Obsidian AI assistant and listed source areas
reviewed during market research.

The detailed note was removed from the public repository before the 2.8.0
license migration to avoid provenance confusion. External commercial products
may be used as market references, but they are not code, license, or
implementation sources for Personal Assistant.

Current rules:

- Do not copy implementation code from external commercial products.
- Do not cite external commercial product internals in public planning docs.
- Record any retained third-party source, template, prompt, or bundled resource
  attribution in `THIRD_PARTY_NOTICES.md` or the relevant release sign-off doc.
- Keep public product strategy language generic unless a named compatibility
  reference is strictly necessary for users.

For the 2.8.0 commercial and license boundary, see
`docs/license-migration-2.8.0.md`.
