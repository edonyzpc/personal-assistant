# Documentation Archive

This directory keeps historical design records, completed SDDs, frozen review
inputs, and superseded trackers. These files are retained for provenance and
evidence only.

Current docs live one level up in [docs/index.md](../index.md). If an archived document conflicts with a current contract or tracker, the current document wins.

Historical files may preserve original prose references such as
`docs/example.md`. When the referenced file is listed in this archive and no
current root-level file exists, resolve that historical path as
`docs/archive/example.md`.
