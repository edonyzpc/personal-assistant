# Third-Party Notices

This file records bundled resources, runtime production dependency licenses,
and runtime license/notice text for Personal Assistant. It is checked from
`package-lock.json`, the bundled-resource provenance table, and the runtime
license notice appendix by:

```bash
npm run check:third-party-notices
```

The runtime dependency table below covers the current production dependency closure reachable
from the root `dependencies` block, including transitive packages and nested
npm package resolutions. Development-only dependencies are excluded.

The local SQLite/WASM Memory backend inlines worker/WASM assets from
`@sqlite.org/sqlite-wasm`; that package is explicitly listed below and must
remain covered whenever the SQLite worker or WASM bundling changes.

Dependency license metadata is taken from lockfile package entries. The package
rows are a release inventory and attribution control; the runtime license
appendix preserves package license/notice files or explicit external notice
sources for packages that do not ship local license files. This file is not a
substitute for formal counsel review before publishing a commercial release.
